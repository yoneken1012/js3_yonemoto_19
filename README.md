# ①課題名
- 怒涛の英単語クイズ

## ②課題内容（どんな作品か）
- 英語学習の効率化を目的にした、ビジネス英単語に特化した4択クイズアプリです。

## ③アプリのデプロイURL
- https://github.com/yoneken1012/js3_yonemoto_19

## ④アプリのログイン用IDまたはPassword（ある場合）
- ID: 今回なし
- PW: 今回なし

## ⑤工夫した点・こだわった点
- 4択問題の自動生成：英単語 + ダミー選択肢3つをランダムに構成する仕組みを実装
- localStorage を使った日別成績の保存：「何問中何問正解」「正答率」を日付ごとに記録
- 学習履歴画面を別ページに切り替えるUI：メイン → 履歴 → メイン の画面遷移を JavaScript で制御
- 制限時間タイマー：1〜10秒の任意設定に対応し、時間切れも自動判定
- 自動で次の問題に進むテンポの良さ：正解 / 不正解 / 時間切れ すべてで約1秒後に次へ
- 一時停止ボタンで学習を中断可能：押している間だけタイマー停止 & 解答ロック

## ⑥難しかった点・次回トライしたいこと（又は機能）
- localStorage のキー管理：日付ごとの記録構造（YYYY-MM-DD）や JSON 読み書きの整理
- ダミー選択肢ランダム生成：正答の重複排除・シャッフル順の保証
- タイマー機能：一時停止、再開、自動次問題遷移などの制御
- 次回トライしたい機能
　・CSVインポートで単語リストを追加
　・出題頻度の重みづけ（間違えた単語を優先出題）
　・単語の音声読み上げ
　・英語 → 日本語、または日本語 → 英語の双方向クイズ
　・スマホ最適化（レスポンシブ対応）
